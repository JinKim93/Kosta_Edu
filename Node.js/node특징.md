## node 
- 크롬 V8 자바스크립트 엔진으로 빌드된 자바스크립트 런타임
- 자바스크립트로 작성된 소스코드를 실행 해줌 -> 서버소스코드면, 서버가 만들어짐 이런느낌

## deno
- typescript 런타임


## node와 비동기성
- 동기방식(Synchronous) : 서버에서 요청을 보냈을 때 응답이 돌아와야 다음 동작을 수행
- 비동기방식(Asynchronous) : 요청을 보냈을때 응답 상태와 상관없이 다음 동작을 수행
