### 준비단계
1. visual studio code 실행
2. 폴더생성(webpage)
3. webpage 폴더에 image(웹페이지에 들어갈 사진)파일 옮김 -> 깃허브 CSS폴더에 올려 놓음

### 첫번째 단계 프레임 구성(index.html)
- index.html 파일생성
 
```html
<html>

    <head>
        <meta charset="utf-8">
        <frameset rows = "30%, 65%, 5%" frameborder = "1">
            <frame src = " "></frame>
            <frameset cols = "20%,80%">
                <frame src = " "></frame>
                <frame src =" " name = "right"></frame>

            </frameset>
            <frame src = " "></frame>
        </frameset>
    </head>

    <body>



    </body>
</html>
```
