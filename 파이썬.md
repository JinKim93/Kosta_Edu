# 파이썬

### print 출력
```py
print("나 혼자 파이썬") 
print(1 + 2)
```

### 변수
```py
value = 100
print(value)
value = "Hello"
print(value)
```

### 정수형 사용하기
```py
width = 10
height = 20
print("사각형의 넓이는",width*height,"이다")
```

### 실수형 사용하기
```py
radius = 5
PI = 3.14
area = radius * radius * PI
print("원의 넓이는",area,"이다")
```

### 문자열 사용하기
```py
print('abcd')
```
- C언어에서 문자,문자열 구분했지만, 파이썬은 구분 안함(문자,문자열 'abcd'로 구분)

### 문자열 데이터
```py
print('a')
print('A')
print('B')
print(ord('a')) #아스키코드 출력
print(ord('A'))
print(ord('B'))
print(chr(65)) #문자로 출력
print(chr(66))
```

### input()함수
```py
strValue = input("좋아하는 과일은 무엇인가요?")
print(strValue + "을좋아합니다")
```
