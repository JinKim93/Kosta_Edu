# 파이썬

### print 출력
```py
print("나 혼자 파이썬") 
print(1 + 2)
```

### 변수
```py
value = 100
print(value)
value = "Hello"
print(value)
```

### 정수형 사용하기
```py
width = 10
height = 20
print("사각형의 넓이는",width*height,"이다")
```

### 실수형 사용하기
```py
radius = 5
PI = 3.14
area = radius * radius * PI
print("원의 넓이는",area,"이다")
```
