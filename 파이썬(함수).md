### 파이썬 함수 (더하기,빼기,곱하기,나누기)

```py

def plus(a,b):
    return a + b

def minus(a,b):
    return a - b

def Mutiple(a,b):
    return a * b

def divide(a,b):
    return a / b

result = plus(10,20)
print(result)

result = minus(10,20)
print(result)

result = Mutiple(10,20)
print(result)

result = divide(10,20)
print(result)

```
