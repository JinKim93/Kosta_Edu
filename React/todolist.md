## TodoList 프로젝트

## 전체구성
1. TodoTemplate
</br>-> 타이틀, 화면 프레임 구성해주는 컴포넌트

2. TodoInsert 
</br>-> 새로운 항목 추가, state 상태 관리

3. TodoListItem
</br>-> 각각의 데이터
</br>-> 각 할일 항목에 대한 정보 컴포넌트

4. TodoList
</br>-> 각각의 데이터 모음
, props 배열,  map을 통해서 TodoListItem 출력
